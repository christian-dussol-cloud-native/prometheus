## About This Repository

This repository contains learning materials and educational examples for **Prometheus**, the open-source monitoring and alerting system graduated by the CNCF.

**Educational focus.** Real code and examples.

### [carousel/](./carousel)

Collection of visual presentations (carousels) illustrating Prometheus concepts.

### [prometheus-first-sample/](./prometheus-first-sample)

Complete hands-on tutorial to get started with Prometheus on Kubernetes, including:

- Prometheus stack deployment (kube-prometheus-stack)
- PromQL queries for cost optimization and waste detection
- Grafana dashboards for cost attribution and resource efficiency
- Kyverno policies for cost governance
- Prometheus + Kyverno governance loop
